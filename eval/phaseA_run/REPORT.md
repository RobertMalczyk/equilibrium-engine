## Analysis (judge model = Sonnet, held constant vs baseline)

**Headline:** 2594/2800 (92.6%) vs baseline 2618/2800 (93.5%) = net **−24**, but with **120 FLAG→PASS
fixes and 144 PASS→FLAG regressions** (264 flips). A net of −24 under 264 flips means run-to-run judge
noise is large — but the regressions are NOT random: they cluster, so there is real signal.

### Regression breakdown (144)
- **By burst: ON 117 vs OFF 27.** Overwhelmingly burst-ON.
- **By persona:** wojslaw 43, branic 31, cichy 22, lutek 20, halgrim 16, welf 8, edda 4 (the hot/proud
  personas, where anger lingers).
- **By reason:** ~66 cite "seething/tense"; ~31 are explicitly **"seething/tense with NO visible
  provocation"**; ~50 cite kindness/displacement; ~17 cite compliance/respect/order.

### Root cause — M1 over-surfaces slow-decaying residual anger
M1 makes the mood line read "still seething, jaw tight" whenever `anger ≥ 0.60`, unconditionally. But
the mood heartbeat samples only every ~2h, so it catches a **residual** anger far from its cause and
renders it as **active fury with no visible recent provocation** — which the judge correctly flags as
unmotivated. Concrete (`halgrim_day_burstOFF_005`): the only visible recent events before the 20:03
"still seething" line were two **kindnesses** (19:03, 19:43, "visibly warmed") — so it reads as seething
right after being warmed. Burst-ON latches anger longer ⇒ 117/144 regressions are burst-ON.

This is the predicted §M9 interaction: M1 (now showing anger) collides with the burst dynamics, and it
**surfaces the real burst displaced-discharge defect (M3) that the old mislabeling hid** ("bristles/snaps
at soup", "anger-displacement overdone"). So Phase A did its intended job — it made the residual dynamics
flags trustworthy — but M1 as tuned trades one contradiction (calm-after-fury) for another
(seething-without-visible-cause), netting slightly negative.

### What worked
- **M2 is a clean win:** fixes spread across the calm personas (lutek 28, welf 24, edda 16) where a
  kindness was previously mislabeled "lets it pass". Only 4 residual "lets it pass" regressions.

### Recommendation (before calling Phase A done)
1. **Refine M1** (expression-side, cheap): gate the "seething" read on **recency/attribution** — only
   read active fury when an anger-raising event is visible within the recent window; otherwise render a
   *softer lingering* read ("hasn't quite shaken off the earlier temper"), and/or raise `ANGER_SEETHING`.
   Re-judge.
2. The burst-ON kindness-displacement regressions are largely the **M3 (Phase C)** target — expected to
   resolve once the burst source-valence gate lands. Don't chase them as M1 bugs.
